# Control de Vigilancia SENA

Este proyecto es el módulo frontend para el control de vigilancia del SENA, desarrollado con Next.js, React y Tailwind CSS.

## Características

- **Registro de Salida**: Control de salida de bienes con validación de firmas.
- **Registro de Reingreso**: Gestión del retorno de bienes.
- **Bitácora**: Historial completo de acciones.
- **Alineación con Base de Datos**: Estructura de datos alineada con PostgreSQL (tablas SOLICITUD, PERSONA, ELEMENTO, etc.).

## Tecnologías

- Next.js 15
- React 19
- Tailwind CSS 4
- Lucide React (Iconos)

## Ejecución

```bash
pnpm dev
# o
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) en su navegador.
